# OnlineBanking #
Online Banking System with most commonly used features

## On the User Frontend ## 

User will be able to 
* Sign up
* Log in
* Deposit
* Withdraw
* Make transaction between two accounts
* Make appointment
etc.

## On the Admin Frontend ## 

Admin will be able to 
* Sign up
* Log in
* Confirm appointment
* Deactivate user account
etc.
